## setTimeout():

```js
setTimeout(function () {
  console.log("Hi");
}, 5000);
// the function is called after 5 seconds
```

## setInterval(): - callback function

- event completes
- action is taken

```js
setInterval(function () {
  console.log("Hi");
}, 2000);
// The function is called for every 2 seconds
```
